Statistical notes

The NCVS acquires its data by asking participants a series of survey questions. Crimes are ranked according to severity, with the most severe crimes, such as assault, asked first. Some incidents can be classified as multiple crimes, in which case the survey classifies it as the more severe crime.

This data tracks the number of individuals who report victimizations, rather than the total number of victimizations.

For these reasons, the figures for some crimes are undercounted.

In 2006, the NCVS changed its methodology and then changed it back in 2007. Thus, estimates from 2006 should not be compared to other years.

Technologies

Data mining and calculations are performed with Python in conjunction with the Pandas, NumPy, and MatplotLib libraries. This site uses the Django web framework with HTML and CSS. Charts for the data visualization tool are rendered with Chart.js.
